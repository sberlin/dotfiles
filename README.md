## bashrc

### Completion r
Adds completion for custom script [r](#r-is-for-run)

Installation:
```sh
$ make complete_r
```

Usage:
```sh
$ r ged[tab][tab]
gedbar          gedfoo          gedit
$ r gedit sam[tab][tab]
sambar.xml      samfoo.log      sample.txt
```

## r is for run
Executable to detach an application completely from shell with nohup.

Usage:
```sh
$ r gedit sample.txt
```
